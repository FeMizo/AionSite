## IMAGEN 1: Portada "5 ERRORES"

### En Canva:

**Paso 1: Crear diseño**
```
Home → Templates
Buscar: "infographic" o "poster"
Seleccionar un template de 1080x1350 (Instagram)
```

**Paso 2: Editar texto principal**
```
Título: "5 ERRORES"
Subtítulo: "que cuesta dinero"
Font: Montserrat Bold (grande) o Poppins Bold

Color: Rojo o naranja llamativo
Fondo: Gris claro o blanco
```

**Paso 3: Agregar números/emojis**
```
Texto grande:
"5️⃣ ERRORES"

Debajo (más pequeño):
"que están matando tus VENTAS 💀"

Abajo del todo:
"Descúbrelo en nuestro blog"
```

**Paso 4: Elementos visuales**
- Agregar líneas divisoras (líneas)
- Agregar iconos: error, stop, warning
- Colores: Rojo (#FF6B6B) + Blanco + Gris

**Paso 5: Exportar**
```
Download → PNG (1080x1350)
Nombre: 5_errores_portada.png
```

### Resultado esperado:
```
┌─────────────────────────┐
│                         │
│    5 ERRORES            │
│  que MATAN tus VENTAS   │
│           💀             │
│                         │
│  Descúbrelo en nuestro  │
│        blog →           │
│                         │
└─────────────────────────┘
```

## ⚡ TEMPLATE RÁPIDO PARA REPET

Una vez hayas hecho una imagen, GUARDARLA COMO TEMPLATE:

1. En Canva → Tu diseño
2. Click "..." (menú superior derecho)
3. Click "Save as template"
4. Nombre: "Portada errores" o similar
5. **Próximo post:** Duplicar template, cambiar números, listo en 2 min
